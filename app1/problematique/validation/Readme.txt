Cette trousse de validation inclut deux catégories d'item:

Maze

Un nouveau labyrinthe vous est fourni avec 4 niveaux de difficulté. 
Afin d'éviter les erreurs d'arrondissement et d'affichage, le fichier Constants.py vous est fourni avec des valeurs ajustées. 

Monster
Un nouveau monstre "beta" est fourni sous le nom Monster_extra_hard.pyc. 
Pour l'utiliser, renommer le "Monster.pyc" et le mettre dans le même dossier que votre main.py.

Pour ce monstre, afin d'éviter les débordements de valeurs, rester dans les valeurs [-MAX_ATTRIBUTE, MAX_ATTRIBUTE].
Votre fitness function devra être beaucoup plus proche de 4 pour effectivement gagner les 4 round du combat.
